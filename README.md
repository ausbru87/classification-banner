# Gnome Desktop Banner


## Installation

### Install System Dependencies


#### Debian Systems
```bash
sudo apt-get update
sudo apt-get install -y python3-gi python3-gi-cairo gir1.2-gtk-3.0 python3-xlib
```


#### Red Hat Systems
```bash
sudo dnf install -y python3-gobject python3-cairo gtk3 python3-xlib
```
